# ERP - Projeto em Grupo

Este repositório contém o código-fonte do nosso projeto de ERP desenvolvido em grupo.  
Abaixo está um guia rápido de como clonar o repositório e contribuir com o projeto usando Git.

---

## 🚀 Clonando o repositório

Antes de começar, certifique-se de ter o **Git** instalado na sua máquina.

```bash
# Clonar o repositório
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git

# Entrar na pasta do projeto
cd SEU-REPOSITORIO

# Criar uma nova branch
git checkout -b nome-da-sua-branch

# Verificar em qual branch você está
git branch

# Adicionar alterações:

# Adicionar todos os arquivos modificados
git add .

# Criar um commit com uma mensagem descritiva
git commit -m "Descrição das alterações"

# Enviar a branch com suas alterações para o GitHub
git push origin nome-da-sua-branch

# Puxando alterações antes de mexer no código:

# Voltar para a branch principal (geralmente "main")
git checkout main

# Puxar as últimas alterações do repositório remoto
git pull origin main
